一个让 AI 自动读取并理解当前项目，生成各大平台爆款文案和矩阵宣发策略的 Skill。

在当前使用目录下的 `nuke` 文件夹生成。

建议搭配中文能力好的模型（如 Deepseek）使用。

*~~注：本项目已实现自举~~*

## Install

下载后，将 `nuke` 文件夹放入你的 skills 文件夹即可。

## Usage

直接启动：
```
/nuke
```

自定义文案数量：
```
/nuke BATCH_COUNT=5
```

## Features
- 从当前项目提取出亮点，优势，并加以宣传。
- 支持同时生成多个文案，形成矩阵宣发。
- 支持小红书，知乎，QQ聊天记录等平台。
- 支持生成 Typst 格式的学术论文，可直达SHIT期刊。
